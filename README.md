# Dye-Count

This is a simple addon that adds a count to the dye crafting station to show how many of each dye you have in your inventory, bank, and warband bank.
![Dye Count](https://github.com/Noshei/DyeCount/blob/main/.preview/DyeCount_preview.png?raw=true)

For any issues, support, or feature requests, please submit an issue on [Github](https://github.com/Noshei/Warband-Bank-Log).
